Welcome to the Effort-Impact Application!


Some instructions to get you started:

Choose Task Sheet contains a list of all available task sheets. Selecting one allows you to either load the associated chart or delete the task sheet.

New Task Sheet allows for the creation of a new task sheet. Input the title of the new sheet (if a sheet with that name already exists - your sheet will have a number appended to the title).

All sheets are initialized with the Task and Description fields. Optional fields can be selected according to the type of tasks that will be added.
