Welcome to the Effort-Impact Application!

Some instructions to get you started:

The first step to generating the Effort-Impact Matrix is selecting your task sheet. All task sheets are located in the Task Sheets folder within the application folder. If you do not yet have a Task sheet in the folder, you can click on the Create New Sheet button, which will create a new Task Sheet with a given name.

Non-optional fields for each entry are: Task, Description. Optional fields can be chosen when creating a new sheet.
