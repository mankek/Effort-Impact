Welcome to the Effort-Impact Application!


Some instructions to get you started:

The left-hand box contains a list of all tables currently available for rendering. Once a table is selected, it can either be opened, which will redirect you to the graph interface, or deleted permanently.

The right-hand box contains a form for creating a new table. Type in the name you wish to give the table (i.e. a project name) and select any of the optional fields you want each task to have. The following characters will be removed from a table name: "?", "/", "\\", "*", ":", "|", "<", ">", "\", ";"

By default, every table has a field for the task name and a task description.
